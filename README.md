# Manajemen Nilai Mahasiswa

## Deskripsi
Program ini merupakan aplikasi berbasis console yang dibuat menggunakan bahasa Python
untuk mengelola data nilai mahasiswa. Program ini menerapkan konsep Pemrograman
Berorientasi Objek (PBO).

## Fitur
- Menambahkan data mahasiswa
- Menampilkan data mahasiswa
- Menghitung rata-rata nilai mahasiswa

## Bahasa Pemrograman
- Python 3

## Cara Menjalankan Program
1. Pastikan Python sudah terinstall
2. Clone repository ini:
   git clone https://github.com/username/manajemen_nilai_mahasiswa.git
3. Masuk ke folder projek
4. Jalankan program dengan perintah:
   python main.py
